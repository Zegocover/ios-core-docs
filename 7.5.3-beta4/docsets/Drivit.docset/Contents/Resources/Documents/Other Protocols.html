  • Flags

    Flags

    The Flags entity is responsible to set the behaviour of the SDK. Flags can be of the following types:

    • Feature Flag: To turn on or off behaviour that is under development.
    • ABTests: To be used when testing A/B© experiments on the SDK
    • Remote Configurations: Variables that the SDK can run e.g.: MobileThreshold, LogLevel, etc.
